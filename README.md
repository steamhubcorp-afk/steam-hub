# SteamHub Project

## Prerequisites
- **Node.js**: This project requires Node.js. Please install it from [nodejs.org](https://nodejs.org/).

## Setup Instructions

### Backend
1. Open a terminal in the `backend` folder.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the server.

### Frontend
1. Open a terminal in the `frontend` folder.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the client.

## Troubleshooting
If you do not have `npm` or `node` installed, the above commands will not work. Please ensure Node.js is installed and added to your system PATH.
